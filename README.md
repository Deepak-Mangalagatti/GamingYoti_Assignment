# GamingYoti_Assignment
Quick-Start Setup Guide
To run and test the application on your machine, follow these steps:
1.	Clone the repository code files to a folder on your computer.
2.	Install the required packages by running: pip install flask requests
3.	Launch the web app server terminal command: python app.py
4.	Open your browser and navigate to the local server port: http://localhost:5000
5.	Fill out the registration inputs, finish the verification workflow, and click 'Get Results'.
